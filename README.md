## Resources

**Display Property** — A CSS property that controls how an element is rendered on the page. Common values include `block` (takes up the full width), `inline` (sits in line with text), and `inline-block` (sits in line but respects width and height).

**Box Model** — The CSS model that describes the space an element takes up. Every element is made up of four layers from inside out: content, padding (space inside the border), border, and margin (space outside the border).

**Responsive Units** — Units of measurement that scale relative to something else rather than being a fixed size. Examples include `vw` (percentage of the viewport width), `vh` (percentage of the viewport height), `%` (percentage of the parent), and `rem` (relative to the root font size).

**Class Selectors** — A way to target HTML elements in CSS using the `class` attribute. A class is defined in HTML with `class="name"` and selected in CSS with `.name`. Multiple elements can share the same class.

**Divs** — A generic block-level HTML element used as a container to group and structure other elements. Divs have no visual style by default and are used to apply layout and styling through CSS.
